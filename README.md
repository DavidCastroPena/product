A collection of internal training resources for product thinking, user research, and customer discovery.

## Contents

### How to Talk to Users
An interactive presentation based on Gustaf Alströmer's (YC Partner) framework for conducting user interviews that generate real insight — not false validation.

**Topics covered:**
- Writing outreach that gets replies
- The golden rules of user interviews
- Questions that unlock insight
- Traps to avoid during calls
- How to synthesize learnings and build an MVP
- How to run sessions after shipping

---

## How to Use

Open the live site link above in any browser. Navigate with the **← →** arrow keys or the on-screen buttons. Works on desktop and mobile.

To run locally, simply open `index.html` in any browser — no server or dependencies required.
